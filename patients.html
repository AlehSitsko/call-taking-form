🛡️ FAQ: Data Privacy and Security

💬 Does this project handle real patient data?
No. This is an educational project only. All patient data used in the system is fictional and provided for demonstration purposes only. This tool is not intended for clinical or commercial use.

🔐 Does this app collect personal information?
No. The project does not collect, store, or transmit any personal data. The database is entirely local and not connected to any external servers or cloud services.

🧠 Why is there a database at all?
The database is included strictly for learning purposes — to demonstrate application architecture, API integration, data validation, and local storage. All actions are available only to the project author, locally.

📄 Is this project HIPAA/GDPR compliant?
No, since it does not process any real personal data. However, the code and design follow security best practices and can be adapted for compliance in future production versions.
